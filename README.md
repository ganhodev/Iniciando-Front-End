# 🚀 Primeiro Projeto de Front-End

Este repositório marca o início da minha jornada no desenvolvimento Front-End.

## Objetivo

Neste momento, meu foco é construir uma base sólida nos fundamentos da web:

* HTML
* CSS
* JavaScript

A ideia é entender como essas tecnologias funcionam na prática antes de utilizar frameworks.

## Próximos passos

Após consolidar os conceitos de HTML, CSS e JavaScript, pretendo evoluir para:

* TypeScript
* Angular

Meu objetivo é aplicar os fundamentos aprendidos em projetos mais robustos, utilizando as ferramentas e práticas do ecossistema moderno de desenvolvimento Front-End.

## Sobre este repositório

Este repositório será atualizado conforme avanço nos estudos, reunindo exercícios, exemplos e pequenos projetos desenvolvidos durante o aprendizado.

---

> Todo desenvolvedor começa pelos fundamentos. Este é o primeiro passo da minha trajetória no Front-End.
